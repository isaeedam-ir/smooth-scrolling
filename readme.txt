== Build Instructions ==

This plugin uses minified JS file for performance.
The original unminified source file are included in the plugin's directory `src`.

The minification process was done using an online tool ([Minify JS Online](https://minify-js.com/)):
- Original file: `js/src/SmoothScroll.js`
- Minified file: `js/SmoothScroll.min.js`

== Installation ==

1. Download the plugin and then Upload the plugin files to the `/wp-content/plugins/smooth-scrolling` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
